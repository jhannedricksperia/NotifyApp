# NotifyApp
Simple Notes App
Notify App — Simple Description

Notify App is a lightweight and user-friendly mobile application designed to help users create, organize, and manage personal notes and reminders. With its clean interface and straightforward features, it allows users to stay on top of their tasks with ease.

Core Features (CRUD Operations)

Create – Add new notes or reminders quickly with a simple and intuitive interface.

Read – View all your saved notes in an organized list for easy access.

Update – Edit existing notes to keep information accurate and up to date.

Delete – Remove notes that are no longer needed with just a tap.
